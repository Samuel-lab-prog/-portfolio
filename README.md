# Portfolio (Astro)

Projeto de portfólio criado no root do monorepo em `portfolio/`, usando Astro.

## Rodar localmente

```bash
cd portfolio
npm install
npm run dev
```

## Scripts

- `npm run dev`: ambiente local
- `npm run build`: build de produção
- `npm run preview`: preview da build
- `npm run check`: validações do Astro
